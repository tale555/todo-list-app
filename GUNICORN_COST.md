# Gunicornの使用料金について

## ✅ 結論：完全無料です！

**Gunicornは無料で使用できます。**

## 📋 詳細説明

### Gunicornのライセンス

- **オープンソースソフトウェア（OSS）**
- **MITライセンス**で公開されている
- **商用利用も含めて完全無料**
- 制限なしで使用可能

### 無料で使える理由

1. **オープンソース**
   - ソースコードが公開されている
   - 誰でも自由に使用・改変・配布できる

2. **MITライセンス**
   - 最も緩いライセンスの一つ
   - 商用利用も含めて制限なし

3. **コミュニティ開発**
   - ボランティア開発者が開発・維持
   - 企業の利益目的ではない

## 💰 コストがかかる場合

### Gunicorn自体は無料

- Gunicornのソフトウェア自体：**無料**

### ただし、本番環境のサーバーには費用がかかる場合がある

Gunicornを動かすための**サーバー**（クラウドなど）には費用がかかる場合があります：

#### 無料で使えるサーバー（例）

1. **Render**（無料プランあり）
   - 無料プラン：制限あり（スリープする）
   - 有料プラン：$7/月〜

2. **Railway**（無料プランあり）
   - 無料プラン：$5分のクレジット/月
   - 小規模アプリなら無料で使える

3. **Heroku**（無料プラン廃止）
   - 以前は無料プランがあったが、現在は有料のみ

4. **PythonAnywhere**（無料プランあり）
   - 無料プラン：制限あり

5. **Fly.io**（無料プランあり）
   - 無料プラン：制限あり

#### 有料のサーバー（例）

1. **AWS（Amazon Web Services）**
   - 従量課金制
   - 小規模なら月数百円程度

2. **Google Cloud Platform**
   - 従量課金制
   - 小規模なら月数百円程度

3. **Azure（Microsoft）**
   - 従量課金制
   - 小規模なら月数百円程度

## 🎯 このアプリの場合

### 開発環境（現在）

- **完全無料**
- 自分のPCで動かすだけ
- サーバー費用なし

### 本番環境（デプロイする場合）

#### オプション1：無料でデプロイ

- **Render**（無料プラン）
- **Railway**（無料プラン）
- **PythonAnywhere**（無料プラン）

→ Gunicornも無料、サーバーも無料

#### オプション2：有料サーバーを使用

- **AWS**、**GCP**、**Azure**など
- 小規模なら月数百円程度

→ Gunicornは無料、サーバーは有料

| **本番環境（有料サーバー）** | サーバー費用のみ |

## 💡 重要なポイント

1. **Gunicorn自体は完全無料**
2. **開発環境は完全無料**
3. **本番環境も無料サーバーを使えば無料**
4. **有料サーバーを使う場合も、Gunicornの費用はかからない**
