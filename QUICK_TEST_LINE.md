# LINE通知のクイックテストガイド

## 📋 テスト手順

### ステップ1: LINE Notifyトークンを取得

1. ブラウザで [LINE Notify](https://notify-bot.line.me/) にアクセス
2. 「LINEにログイン」をクリック
3. 右上の「マイページ」をクリック
4. 「トークンを発行する」をクリック
5. 以下の情報を入力：
   - **トークン名**: 「Todo List通知」など任意の名前
   - **通知を送信するトークルームを選択**: 「1-on-1でLINE Notifyから通知を受け取る」を選択
6. 「発行する」をクリック
7. **表示されたトークンをコピー**（重要：後で確認できません）

### ステップ2: .envファイルを作成

`todo_list_app`ディレクトリに`.env`ファイルを作成し、以下の内容を記入：

```env
LINE_NOTIFY_TOKEN=ここに取得したトークンを貼り付け
```

例：
```env
LINE_NOTIFY_TOKEN=abcdefghijklmnopqrstuvwxyz1234567890
```

### ステップ3: テストを実行

#### 方法1: コマンドラインからテスト

```bash
cd todo_list_app
python test_line_notify.py
```

#### 方法2: ブラウザからテスト

1. アプリケーションを起動：
   ```bash
   python main.py
   ```

2. ブラウザで `http://localhost:5000` にアクセス

3. 画面上部の「📱 LINE通知送信」ボタンをクリック

## ✅ 成功の確認

- コマンドラインで「✅ テスト通知の送信に成功しました！」と表示される
- LINEアプリに通知が届く

## ❌ エラーが発生した場合

- トークンが正しく設定されているか確認
- `.env`ファイルが`todo_list_app`ディレクトリに正しく配置されているか確認
- トークンに余分なスペースや改行が含まれていないか確認
