# RenderのAdvanced設定について

## 📋 各項目の説明

### 1. Secret Files（秘密ファイル）

**必須**: ❌ 不要

- 環境変数で設定しているため、この項目は不要です
- そのままにしておいてOK

### 2. Health Check Path（ヘルスチェックパス）

**必須**: ❌ 不要（デフォルトでOK）

- デフォルト: `/healthz`
- このアプリにはヘルスチェックエンドポイントがないため、そのままにしておいてOK
- または、空欄にしても問題ありません

### 3. Pre-Deploy Command（デプロイ前コマンド）

**必須**: ❌ 不要

- データベースマイグレーションなどが必要な場合に使用
- このアプリでは不要なので、空欄のままでOK

### 4. Auto-Deploy（自動デプロイ）

**必須**: ✅ デフォルトでOK

- デフォルト: "On Commit"
- GitHubにプッシュするたびに自動的にデプロイされます
- そのままにしておいてOK

### 5. Build Filters（ビルドフィルター）

**必須**: ❌ 不要

- 特定のパスの変更のみでデプロイをトリガーしたい場合に使用
- このアプリでは不要なので、そのままにしておいてOK

## ✅ 推奨設定

**すべてデフォルトのままでOKです！**

特に変更する必要はありません。そのまま「Deploy Web Service」ボタンをクリックしてください。

## 🚀 次のステップ

1. 「Deploy Web Service」ボタンをクリック
2. デプロイが開始されます（5-10分かかります）
3. デプロイが完了したら、提供されたURLにアクセス

## ⚠️ 注意事項

- デプロイ中は環境変数の設定を変更しないでください
- デプロイが完了するまで待ってから、動作確認を行ってください
