# テストガイド

## 🧪 修正内容のテスト方法

### 1. 削除ボタンのサイズ統一のテスト

#### テスト方法

1. **一覧画面にアクセス**
   - `https://todo-list-app-h6za.onrender.com/simple_list` にアクセス
   - または、アプリの「一覧」ボタンをクリック

2. **ボタンサイズを確認**
   - 編集・削除・完了ボタンが同じサイズになっているか確認
   - すべてのボタンが同じ高さ・幅になっているか確認

3. **レスポンシブ表示で確認**
   - ブラウザの開発者ツール（F12）を開く
   - デバイスモード（Ctrl+Shift+M）を有効にする
   - スマートフォンサイズ（例: iPhone 12）に変更
   - ボタンサイズが統一されているか確認

#### 確認ポイント

- ✅ 編集ボタンと削除ボタンが同じサイズ
- ✅ 完了ボタンも同じサイズ
- ✅ レスポンシブ表示でもサイズが統一されている

### 2. レスポンシブ時のドラッグ&ドロップのテスト

#### テスト方法A: ブラウザの開発者ツールでテスト

1. **開発者ツールを開く**
   - F12キーを押す
   - または、右クリック → 「検証」

2. **デバイスモードを有効にする**
   - Ctrl+Shift+M（Windows）
   - Cmd+Shift+M（Mac）
   - または、ツールバーのデバイスアイコンをクリック

3. **デバイスを選択**
   - iPhone 12、iPad、Galaxy S20など
   - または、カスタムサイズを設定

4. **ドラッグ&ドロップをテスト**
   - Todoアイテムを長押し（右クリック）
   - ドラッグして移動
   - 順番が変わるか確認

#### テスト方法B: 実際のモバイルデバイスでテスト

1. **スマートフォン/タブレットでアクセス**
   - `https://todo-list-app-h6za.onrender.com` にアクセス

2. **ドラッグ&ドロップをテスト**
   - Todoアイテムを長押し
   - ドラッグして移動
   - 順番が変わるか確認

#### テスト方法C: テスト用HTMLファイルを使用

1. **テストファイルを開く**
   - `test_responsive_drag.html` をブラウザで開く

2. **テストを実行**
   - デスクトップ: マウスでドラッグ&ドロップ
   - モバイル: 長押ししてドラッグ
   - ログを確認

#### 確認ポイント

- ✅ デスクトップでマウスドラッグが動作する
- ✅ モバイルでタッチドラッグが動作する
- ✅ ボタンをクリックしてもドラッグが開始されない
- ✅ 順番が正しく保存される

## 🔍 デバッグ方法

### ブラウザの開発者ツールで確認

1. **コンソールを開く**
   - F12 → 「Console」タブ

2. **エラーを確認**
   - エラーメッセージがないか確認
   - 警告メッセージがないか確認

3. **ネットワークを確認**
   - 「Network」タブで、`/update-order`へのリクエストが送信されているか確認

### タッチイベントのログを確認

ブラウザのコンソールに以下を入力：

```javascript
// タッチイベントのログを有効にする
document.addEventListener('touchstart', function(e) {
    console.log('touchstart:', e.touches[0].clientY);
}, true);

document.addEventListener('touchmove', function(e) {
    console.log('touchmove:', e.touches[0].clientY);
}, true);

document.addEventListener('touchend', function(e) {
    console.log('touchend');
}, true);
```

## 🐛 トラブルシューティング

### ドラッグが動作しない場合

1. **ブラウザのコンソールでエラーを確認**
2. **タッチイベントが有効か確認**
   - 開発者ツールでデバイスモードが有効になっているか確認
3. **JavaScriptが有効か確認**
   - ブラウザの設定でJavaScriptが有効になっているか確認

### ボタンサイズが統一されていない場合

1. **ブラウザのキャッシュをクリア**
   - Ctrl+Shift+Delete（Windows）
   - Cmd+Shift+Delete（Mac）
2. **ハードリロード**
   - Ctrl+F5（Windows）
   - Cmd+Shift+R（Mac）
